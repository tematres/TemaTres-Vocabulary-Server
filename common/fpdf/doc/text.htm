Text

Text(float x, float y, string txt)

Descripción

Imprime una cadena de carácteres. El origen es a la izquierda del primer carácter, sobre la línea base. Este método permite colocar una cadena precisamente en la página, pero es usualmente más fácil usar Cell(), MultiCell() o Write() los cuales son los métodos estándades para imprimir texto.

Parámetros

x
Abscisa del origen.
y
Ordenada del origen.
txt
Cadena a ser impresa.

Vea además

SetFont, SetTextColor, Cell, MultiCell, Write
